# ブログを開設しました

こんにちは、yasukogaです。

このたび、ポートフォリオサイトにブログ機能を追加しました。

## このブログについて

このブログでは、以下のようなトピックについて書いていく予定です：

- **プロジェクトマネジメント**: PMとしての経験や学び
- **フロントエンド技術**: 元Frontend Engineerとしての技術的な知見
- **チーム開発**: チームビルディングやコミュニケーションについて
- **個人的な学び**: 日々の気づきや挑戦

## 技術スタック

このブログは以下の技術で構築されています：

- **静的サイト生成**: Node.jsでMarkdownからHTMLを生成
- **ホスティング**: GitHub Pages
- **デザイン**: シンプルなHTML + CSS

```javascript
// ビルドスクリプトで記事を生成
npm run build:blog
```

## これから

定期的に記事を投稿していきたいと思います。
プロジェクトマネジメントやフロントエンド開発に興味がある方の参考になれば嬉しいです。

よろしくお願いします！
